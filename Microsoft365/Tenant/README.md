# Microsoft 365 - Tenant

Tenant-wide administration spanning more than one workload.

**Belongs here:** licence assignment and reporting, service health, tenant configuration, secure score, and cross-workload reporting.

**Does not belong here:** anything scoped to a single workload. A mailbox script is [Exchange](../Exchange/README.md) even when reached through a tenant-level API.
