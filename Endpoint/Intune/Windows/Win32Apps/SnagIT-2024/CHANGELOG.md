## [1.1.0] - 2026-07-23

### Changed

- Relocated from `Intune/Windows/Win32Apps/SnagIT 2024` to `Endpoint/Intune/Windows/Win32Apps/SnagIT-2024` as part of the repository restructure. Raw URLs built on the previous path no longer resolve; the pre-reorg state is preserved at tag `v1.0-pre-reorg`.
- Renamed `install-snagit-2024.ps1` to `Install-Snagit2024.ps1` to match the approved `Verb-Noun` naming convention.
- Renamed `uninstall-snagit-2024.ps1` to `Uninstall-Snagit2024.ps1` to match the approved `Verb-Noun` naming convention.
- Repointed the repository URLs in the script headers following the GitHub account rename from `bigjoestretch` to `joelcottrell`.
