# VMware PowerCLI - Reporting

Read-only inventory and reporting across vSphere.

**Belongs here:** inventory exports, capacity and utilisation reporting, configuration audits, and drift detection.

**Does not belong here:** anything that writes. A script that reports and then remediates belongs with the thing it changes, not here. Keeping this folder read-only means it can be run safely against production without review.
