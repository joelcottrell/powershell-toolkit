# Windows - Server

Windows Server administration: roles, features, and server-only surfaces.

**Belongs here:** role and feature installation, Active Directory Certificate Services, DHCP, DNS, IIS, failover clustering, and server deployment.

**Does not belong here:** anything that also runs on a workstation. If it runs unmodified on both, it belongs in [Common](../Common/README.md).
