# Windows - Client

Windows workstation administration and user experience.

**Belongs here:** shell and taskbar configuration, user profile settings, client application management, and workstation-only features such as WSL.

**Does not belong here:** anything that also runs on a server. If it runs unmodified on both, it belongs in [Common](../Common/README.md).
